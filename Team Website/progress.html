Examples/Trials made

  • Building the front-end proved to be mainly issue free.
  • Writing the back-end scripts were difficult due to issue with Drupal REST JSON parsing errors and token requests.
  • We ran the app on the Phonegap Developer app, after building it with the responsive design mode in Safari.
  • We built the app with Phonegap for Android and attempted to run it, however front-end was unresponsive due to not using highdpi mode.
  • AJAX Requests and Javascript were not loading on the Phonegap Build Android app. The fix was to move the Javascript in the same folder as index.js.
  • Built the app with Hydration mode, AJAX Requests work.
  • Removing Hydration causes the AJAX Requests to stop working. Workaround for this was to reinstall Cordova Whitelist plugin for the REST endpoint. (Does not work, known issue with Phonegap)
  • Built the app with Phonegap CLI for iOS, inAppBrowser was not working. The fix was to add it via Phonegap CLI. (The app fully works on iOS).
  • Added notifications plugin with PushWoosh for Android, as iOS requires developer account. Phonegap build would simply replace the app with the plugin. (Known issue)
