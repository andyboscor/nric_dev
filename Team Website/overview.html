Background and Problem

The International Resource for Infection Control (iNRIC) represents a very important part of healthcare that is used across the world by various medical staff due to itâ€™s extensive database of articles, news and links gathered from WHO, ECDC and many other sources. The main website (nric.org.uk) is currently used and continuously updated with information.

The website is responsive and can work on mobile platforms, however it does not allow any sort of personalised customisation for each individual user. Furthermore, underdeveloped countries are unlikely to be able to access the desktop version, so they would need an easier to use app than the mobile version of the website.
